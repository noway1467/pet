# 桌面宠物 (Desktop Pet) v3.9.3

这是一个功能丰富、高度可定制的 Windows 桌面宠物应用程序，基于 [PySide6](https://wiki.qt.io/Qt_for_Python) 开发。它不仅能在你的桌面上提供一个透明、无边框、可置顶的虚拟陪伴，还内置了物理引擎、多种宠物形态（像素、自定义图片、Live2D）、丰富的交互反馈、情感养成系统，以及结合系统 TTS 的语音播报功能。

## ✨ 核心特性

### 1. 多样化的渲染引擎
本项目支持三种不同类型的宠物形态，随时可以通过右键菜单或系统托盘进行切换：
* 👾 **像素宠物 (Pixel Pet)**：内置如史莱姆 (Slime)、猫咪 (Cat) 等程序化动画角色，支持“像素风”和“平滑风”两种渲染模式。
* 🖼️ **图片宠物 (Image Pet)**：支持加载本地的静态图片或动图 (PNG, JPG, WEBP, GIF 等)，支持水平镜像翻转，并可配置面部特征以实现视线跟随。
* 🎭 **Live2D 模型 (可选)**：支持完整的 Live2D 动态模型渲染（兼容 Cubism 2 的 `model.json` 和 Cubism 3 的 `.model3.json`），支持模型自带动作触发与视线跟随。

### 2. 沉浸式桌面交互
* **无边框与透明背景**：完美融入桌面环境。
* **物理重力引擎**：鼠标拖拽松手后，宠物会受重力影响自然掉落到底部或任务栏上方。
* **边缘吸附与隐藏**：将宠物拖至屏幕边缘（左、右、上）可自动吸附并缩回隐藏，鼠标靠近时会自动探出。
* **动作反馈**：支持通过点击触发内置的窗口动作（如起跳、蹦跳、点头、摇头、转身、跳舞等）。
* **防遮挡机制**：支持设置“鼠标穿透”和“避开任务栏”。

### 3. 伴侣、雌小鬼与养成系统 (Personality & Affinity)
通过内置的配置系统，你可以让宠物拥有不同的“性格”：
* **普通气泡**：宠物会根据设定的频率随机弹出日常语录。
* **伴侣模式 (Companion Mode)**：开启后，气泡内容将切换为情侣专属语录（告白、情话、打情骂俏等）。
* **雌小鬼模式 (Mesugaki Mode)**：开启后，宠物会以傲娇、调侃的语气与你互动。
* **养成模式 (Nurture Mode)**：引入好感度系统，随着陪伴时长的增加、日常的互动（如摸头），好感度会逐渐提升，解锁不同阶段的专属台词。
* **节日问候**：支持配置用户生日与自定义纪念日，在特定日子宠物会送上专属问候。

### 4. 语音与 TTS 朗读
* **Live2D 原生语音**：在触发特定动作（如摸头、戳身体）时，播放模型自带的 `.wav` 语音。
* **TTS 系统朗读**：基于系统自带的 Text-To-Speech 服务 (QtTextToSpeech)，支持将气泡文本用语音读出来。支持调整音量、语速，甚至接入自定义的 TTS 后端命令。

## 🚀 安装与运行

### 1. 环境要求
* **操作系统**：Windows 10 / 11
* **Python 版本**：Python 3.8 或更高版本

### 2. 依赖安装
在项目根目录下打开终端，执行以下命令安装基础依赖：

```bash
pip install -r requirements.txt
```

核心依赖项包括：
* `PySide6 >= 6.6` （UI 框架与核心功能）
* `pygame >= 2.0.0` （音频与音效播放支持）
* `send2trash >= 1.8.0` （文件回收站操作支持）

### 3. 可选功能配置 (Live2D 支持)
> **⚠️ 注意：Live2D 为可选功能。**
如果你不需要 Live2D 模型，项目依然可以以像素宠物或图片宠物完美运行。

如果需要加载 Live2D 模型，你需要额外安装 `live2d-py` 库：
```bash
pip install live2d-py
```
**配置模型数据：**
将你的 Live2D 模型文件夹放置在项目根目录的 `live2d/` 文件夹下（或通过设置自定义的模型路径）。如果是常用的模型，可以将其放入 `live2d/常用/` 目录下，以便在应用内的模型选择器中置顶显示。

### 4. 启动应用
安装完依赖后，运行以下命令启动桌面宠物：
```bash
python main.py
```
你也可以直接双击项目目录下的 `启动桌面宠物.bat` 来启动。

## 🛠️ 打包与发布

如果你希望将项目打包成独立的 `.exe` 可执行文件，以便在没有 Python 环境的电脑上运行：
1. 确保已安装 `pyinstaller`。
2. 双击运行 `build_exe.bat` 脚本。
3. 编译完成后，生成的程序会位于 `dist/` 文件夹中。

*(注：由于 PyInstaller 打包逻辑，如果需要附带 Live2D 预设模型，请确保 `live2d/` 文件夹位于 `dist/` 对应的相同层级，或通过配置文件指定路径。)*

## 📂 主要文件结构说明
* `main.py` - 主程序入口，包含窗口逻辑、系统托盘、边缘吸附与手势判断。
* `config.py` - 配置管理器，负责读取和保存用户设置（配置保存在 `~/.desktop-pet/config.json`）。
* `pixel_pet.py` / `image_pet.py` / `live2d_pet.py` - 不同形态宠物的具体渲染引擎。
* `chat_bubble.py` - 气泡系统逻辑及界面。
* `affinity.py` - 好感度与养成系统核心逻辑。
* `*_quotes.py` - 包含普通语录、伴侣语录、雌小鬼语录等文本数据。
* `tts_player.py` / `voice_player.py` - 系统 TTS 以及音频播放控制模块。

## 📝 许可与归属
* 本项目的 UI 框架基于 **PySide6** 构建。
* Live2D 相关的解析和渲染依赖于开源社区的 `live2d-py` 扩展。
* 项目中附带的部分内置图像、模型素材版权归原作者或原出处所有，仅供学习、测试使用。
